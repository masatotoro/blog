Org-modeにHTMLを埋め込む方法

Date 2019-03-03 (日) Tags Emacs / Org-mode

目次

  • Org-mode 9.0の変更点
  • HTMLエクスポート時の問題点
  • 解決方法

　EmacsのOrg-modeは、HTMLにエクスポートする機能がある。 Org-modeには、HTMLのタグを直書きするために、以下のブロックが用意されている。 

#+BEGIN_HTML
#+END_HTML

Org-mode 9.0の変更点

　Org-mode 9.0の変更点の中に、以下の項目がある。

図1: エクスポートブロックの新しいシンタックス

HTMLエクスポート時の問題点

　The HTML export back-end transforms ‘<’ and ‘>’ to ‘&lt;’ and ‘&gt;’.

  • https://orgmode.org/manual/Quoting-HTML-tags.html

解決方法

org-insert-structure-template 

Select a key
============

[a]     export ascii
[c]     center
[C]     comment
[e]     example
[E]     export
[h]     export html
[l]     export latex
[q]     quote
[s]     src
[v]     verse
[       ]     Press TAB, RET or SPC to write block name
