Goemonとは何者か

　Goemonとは、シンプルで使いやすいタスクランナーである。 監視対象のファイルの更新を検知し、指定しているタスクを実行し、 ライブリロードが出来る。

Go言語のインストール

　Goemonを利用するために、Go言語をインストールする。 

sudo apt install -y golang

　インストールできたバージョンを確認する。 

$ go version
go version go1.10.4 linux/amd64

Goemonのインストール

　以下のコマンドを実行すると、インストールができる。 

go get github.com/mattn/goemon/cmd/goemon
